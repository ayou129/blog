NodeJs

创建于：2019年4月3日上次更新：2021年10月8日

node使用较少，但是有必要总结一下，以下是总结详情

安装Nodejs

https://nodejs.org/en/download/current/

  1. cd /usr/src/ && wget https://nodejs.org/dist/v12.18.1/node-v12.18.1-linux-x64.tar.xz
  2. tar xf node-v12.18.1-linux-x64.tar.xz
  3. mkdir /usr/local/node && mv node-v12.18.1-linux-x64/* /usr/local/node
  4. 修改系统变量 PATH=$PATH:/usr/local/node/bin
  5. node -v
  6. sudo ln -s /usr/local/node/bin/node 这样可以执行 sudo node
  7. sudo ln -s /usr/local/node/bin/npm 这样可以执行 sudo npm

NodeJs文件

Nrm

  1. 安装nrm npm i -g nrm

  2. 测试所有 源 nrm test

  3. 选择最快的 源 nrm use *

  4. 完毕
