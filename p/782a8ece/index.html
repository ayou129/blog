生成秘钥

  1. cd ~/.ssh
  2. ssh-keygen -t rsa -C “guoxinlee129@gmail”
  3. 接着按3个回车
  4. 在c/Users/Administrator/.ssh生成了加密文件，复制id_rsa.pub
  5. 打开github 添加New SSH key
  6. 验证是否连接成功 ssh -T git@github.com(gitee site)
