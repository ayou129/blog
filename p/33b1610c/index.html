Ubuntu核心知识要点

发表于 2016-04-26 更新于 2022-09-28 分类于 编程语言
学习Ubuntu核心知识要点

个人使用CentOS比较多，Ubuntu作为拓展学习一下。

前言

下载链接 https://ubuntu.com/download/alternative-downloads

问题

  1. 没有root密码

    • 解决方案：
      1. sudo passwd
      2. 输入用户的密码
      3. 输入root的初始化密码
      4. 重复输入

  2. 刚安装好的系统 出现 Unable to fetch some archives, maybe run apt-get update or try with --fix-missing?

    • 解决方案：
      1. sudo vim /etc/resolv.conf
      2. 添加nameserver 223.5.5.5,8.8.8.8
      3. 执行 sudo apt-get update

  3. 无法登录ROOT账号

    1. su root
    2. sudo gedit /etc/lightdm/lightdm.conf
    3. 添加
      • greeter-show-manual-login=true #手工输入登陆系统的用户名和密码
      • allow-guest=false #不允许guest登录
    4. reboot

LVS

概念

Linux Virtual Server的简写，即Linux虚拟服务器，是一个虚拟的服务器集群系统

使用的目的

通过LVS提供的负载均衡技术和Linux操作系统实现一个高性能、高可用的服务器集群，具有良好可靠性、可扩展性和可操作性

实现原理

虚拟服务器所有的方法

  1. 基于DNS域名轮流解析的方法
  2. 基于客户端调度访问的方法
  3. 基于应用层系统负载的调用方法
  4. 基于IP地址的调度方法(LVS使用它)

IP负载均衡技术是通过IPVS内核模块来实现的，IPVS是LVS集群的核心软件
安装在Director Server上，同时在Director Server上虚拟出一个IP(Virtual IP)地址

访问流程：域名访问 -> 域名解析 解析到 IP -> Director Server
访问的请求首先经过 Virtual IP 到达负载调度器
由负载调度器从 真实 服务器列表中选取一个服务器借点响应用户的请求

Real Server借点如何返回数据给用户：是IPVS实现的重点技术
IPVS实现负载均衡极致有三种方法

  1. NAT 地址转发
  2. TUN IP ** 模式
  3. DR 直接路由

使用LVS优点：
开源免费
技术资源很丰富
