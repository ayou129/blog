甜点s

菜品

家常菜

青椒炒蛋

食材准备

  1. 鸡蛋打入后，添加 盐、胡椒粉、白醋(提鲜) ,一个方向搅拌2分钟，使空气充分进入蛋液中，不容易煎糊
  2. 青椒切碎，这样易熟
  3. 蒜、姜、葱

开始制作：

  1. 油滑锅，但是油不能太热
  2. 炒鸡蛋，25-40s，捞出
  3. 青椒、葱姜蒜下锅 50s，大火
  4. 倒鸡蛋
  5. 食用盐、白糖、生抽酱油、胡椒粉

花菜

葱姜蒜+老干妈+花菜，注意：花菜要切小个

  1. 中火，倒油
  2. 倒入葱姜蒜+老干妈
  3. 倒入花菜，1分钟后，放生抽、醋、盐
  4. 放一点水，盖上盖子焖一下，不然不熟
  5. 放盐、胡椒粉调味
  6. 出锅

炒土豆

葱姜蒜+土豆

  1. 中火，倒油
  2. 倒入土豆，1分钟后，放生抽、醋、料酒(提鲜)
  3. 放盐、胡椒粉调味
  4. 出锅

鸡蛋饼

鸡蛋 + 面粉 + 盐 + 小葱或大葱，快速搅拌防止有大面泡

  1. 中火，倒油
  2. 每个饼翻四次
  3. 出锅
