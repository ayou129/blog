    防止跳帧

    一段视频中间删掉n个帧后会发生跳帧

    操作：视频过度 -> 溶解 -> MorphCut -> 拖动
    如果有瑕疵，可以在时间线面板 右键 设置过度持续时间大一些

    红色状态：正在处理
    绿色：处理成功

    快速对齐音视频

    在项目中选中音频&&视频 右键 -> 合并剪辑

    标记

    两种标记(M键)

    1. 适用于采访类节目(单个项目总时间固定)
      针对音视频的标记

    2. 适用于新闻类节目(总时间、单个项目时间固定)
      针对时间线的标记

    标记可以导出，方便沟通吸纳意见
    操作：文件 -> 导出 -> 标记

    批量只做字幕

    编写字幕txt

    1. 第一行，字幕标题
    2. 后面每一行都是一次字幕

    使用 PS：

    1. 新建
    2. 文档类型：胶片和视频
    3. 背景：透明
    4. 增加文字 样式自定义(大小、对齐方式、颜色、描边等)
    5. 图像 -> 变量 -> 定义 -> 文本替换：写上字幕标题
    6. 点击下一个
    7. 导入字幕txt文件，编码utf8
    8. 文件 -> 导出 -> 数据组作为文件 -> 选择文件夹 -> 确定

    使用 Premiere：

    1. 导入时，右键选中所有字幕右键，设置持续时间(默认是5秒)
    2. 拖动时间线

    制作(静态)分屏效果

    方式1：裁剪

    1. 缩小所有素材并且摆放
    2. 视频效果 -> 变换 -> 裁剪 视频的边缘，制作黑色边框(分屏效果)

    方式2：遮罩

    分屏效果使用场景

    1. 素材量比较大 但是质量参差不齐
    2. 视频有核心主题(线索) 贯穿始终
    3. 可以做强调效果
    4. 表达同一时间不同空间发生的事情(平行蒙太奇)
    5. 阿卡贝拉 纯人声

    扩展：局部去色

    制作(动态)分屏效果

    选中A视频

    1. 裁剪
    2. 位置
    3. 选方向并打关键帧
    4. shift + 方向右键 每按一次平移5帧

    前缓出、后缓入

    保留视频单一颜色

    快速删除视频间隙

    1. 序列菜单
    2. 封闭间隙

    制作电影职位表字幕

    方式1：内置功能：滚动字幕
    方式2：PS制作图片，位移动画功能
    方式3：

    1. 在节目面板中增加文字工具，把txt文本复制进去
    2. 把文字区域拉满，显示全部的字幕
    3. 水平、垂直居中对齐
    4. 增加滚动效果：基本图形面板 -> 编辑 -> 滚动
    5. 通过文字图层的长度决定

    音频降噪 && 修复

    1. 选中premiere中的音频
    2. 右键 -> 在 audition 中编辑
    3. 找出有噪音的地方
    4. 效果 -> 降噪/恢复 -> 降噪(处理)
    5. 捕捉 噪声样本
    6. 降噪 调低百分比(50% - 60%)
    7. 可以重复降噪 ctrl + R

    消除所有声音：

    1. 选中某段声音
    2. 效果 -> 静音

    声音更大更稳定：

    1. 效果组
    2. 振幅与压限
    3. 语音音量级别
    4. 目标音量级别调至 -15
    5. 电平值 50

    去除部分口水音

    1. 工具栏 -> 污点修复画笔工具
    2. 涂抹即可
