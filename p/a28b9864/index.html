基础命令

系统安全

  • sudo 以系统管理者的身份执行指令
  • su 用于变更为其他使用者的身份

文件管理

  • chmod 控制文件的权限

    • u 表示该文件的拥有者，g 表示与该文件的拥有者属于同一个群体(group)者，o 表示其他以外的人，a 表示这三者皆是。
      • 表示增加权限、- 表示取消权限、= 表示唯一设定权限。
    • r 表示可读取，w 表示可写入，x 表示可执行，X 表示只有当该文件是个子目录或者该文件已经被设定过为可执行。

  • setfacl 可以更精确的控制权限的分配。

进程管理

  • w 显示目前登入系统的用户信息。

  • top 实时显示 process 的动态。 https://www.runoob.com/linux/linux-comm-top.html

  • ps 显示当前进程 process 的状态。

  • kill 用于删除执行中的程序或工作。

    • @param [-s <信息名称或编号>][程序]　或　kill [-l <信息编号>]

  • pkill 终止进程，按终端号踢出用户

信号编号 信号名 含义
0 EXIT 程序退出时收到该信息。
1 HUP 挂掉电话线或终端连接的挂起信号，这个信号也会造成某些进程在没有终止的情况下重新初始化。
2 INT 表示结束进程，但并不是强制性的，常用的 “Ctrl+C” 组合键发出就是一个 kill -2 的信号。
3 QUIT 退出。
9 KILL 杀死进程，即强制结束进程。
11 SEGV 段错误。
15 TERM 正常结束进程，是 kill 命令的默认信号。
  • pstree 将所有行程以树状图显示
  • killall 根据名字来杀死进程，它会给指定名字的所有进程发送信息。

问题：有/无pid，如何杀掉一个进程

用户管理

  • id 显示用户的ID，以及所属群组的ID。
  • usermod 用于修改用户帐号。
  • useradd 用于建立用户帐号。
  • groupadd 用于创建一个新的工作组，新工作组的信息将被添加到系统文件中。
  • userdel 用于删除用户帐号。加 -r 连同用户文件一起删除

文件系统

  • mount 用于挂载Linux系统外的文件。
  • umount 用于卸除文件系统。
  • fsck 用于 检查与修复 Linux 档案系统，可以同时检查一个或多个 Linux 档案系统。
  • df 用于显示目前在Linux系统上的文件系统的磁盘使用情况统计。
  • du 用于显示目录或文件的大小。

####系统关机和重启

  • shutdown 用来进行关机程序，并且在关机以前传送讯息给所有使用者正在执行的程序，shutdown 也可以用来重开机。
  • reboot 重新启动计算机。

网络应用

  • curl 利用URL规则在命令行下工作的文件传输工具
  • telnet 用于远端登入。
  • mail 邮件命令
  • elinks 文本编辑器

网络测试

  • ping 用于检测主机
  • netstat 用于显示网络状态。
  • host 测试域名系统工作是否正常。

网络配置

  • hostname 查看更改hostname命令解析
  • ipconfig 网络配置命令

常用工具

  • ssh 远程连接工具 https://www.cnblogs.com/ftl1012/p/ssh.html
  • screen 由GNU计划开发的用于命令行终端切换的自由软件。 https://idc.wanyunshuju.com/cym/68.html
  • clear 用于清除屏幕。
  • who 用于显示系统中有哪些使用者正在上面，显示的资料包含了使用者 ID、使用的终端机、从哪边连上来的、上线时间、呆滞时间、CPU 使用量、动作等等。
  • date 显示或设定系统的日期与时间

软件包管理

  • yum 一个在Fedora和RedHat以及SUSE中的Shell前端软件包管理器。基於RPM包管理，能够从指定的服务器自动下载RPM包并且安装，可以自动处理依赖性关系，并且一次安装所有依赖的软体包，无须繁琐地一次次下载、安装。
  • rpm RPM软件包的管理工具。
  • apt-get deb包管理式的操作系统，主要用于自动从互联网的软件仓库中搜索、安装、升级、卸载软件或操作系统。

文件查找和比较

  • find 用来在指定目录下查找文件。 https://www.runoob.com/linux/linux-comm-find.html

系统定时任务

开启定时任务

  • crontab

    • crontab -e #编写定时任务 命令(分时日月周)
    • 在以上各个字段中，还可以使用以下特殊字符：
    • 星号（*）：代表所有可能的值，例如month字段如果是星号，则表示在满足其它字段的制约条件后每月都执行该命令操作。
    • 逗号（,）：可以用逗号隔开的值指定一个列表范围，例如，“1,2,5,7,8,9”
    • 中杠（-）：可以用整数之间的中杠表示一个整数范围，例如“2-6”表示“2,3,4,5,6”
    • 正斜线（/）：可以用正斜线指定时间的间隔频率，例如“0-23/2”表示每两小时执行一次。同时正斜线可以和星号一起使用，例如*/10，如果用在minute字段，表示每十分钟执行一次。
