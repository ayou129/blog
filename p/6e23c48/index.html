编辑 host

  • Mac：sudo vim /etc/hosts
    • 刷新 dscacheutil -flushcache
      • Mac OS X 10.7 – 10.8：sudo killall -HUP mDNSResponder
      • Mac OS X 10.5 – 10.6：dscacheutil -flushcache
      • Mac OS X 10.4：lookupd -flushcache
  • Windows
    • 刷新 ipconfig /flushdns

Done！
